# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024

### Added
- Initial release of the Temple Donation SPA
- Multi-language support (6 languages: EN, HI, TE, MR, BN, OD)
- UPI QR code donation functionality
- Bank transfer details section
- WhatsApp payment confirmation integration
- Google Sheets integration for donor records
- Real-time donation progress tracking
- Animated counters for statistics
- Responsive design for mobile, tablet, and desktop
- PWA (Progressive Web App) support with manifest and icons
- Temple comparison section (before/after)
- Trust and transparency section
- Temple committee information
- Scrolling notice bar for important announcements
- Donor list modal with categorized view (Confirmed/Pending)
- Traditional Indian temple aesthetics with saffron, gold, and crimson theme
- Mandala background pattern
- Floating particle animations in hero section
- Decorative divider with Om symbol
- Smooth scroll navigation
- Language selector dropdown

### Technical
- Built with React 19 and Vite
- Uses @vitejs/plugin-react-swc for fast compilation
- CSS-in-JS styling approach
- Intersection Observer API for scroll animations
- Google Sheets CSV parsing for donor data
- Mobile-first responsive design
- Single-file component architecture
